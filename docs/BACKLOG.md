## 기능 (중요도 순)

### 1. 웰컴 키트 미리보기
배송지 폼 위에 실제 키트 구성 이미지 배치.
"무엇을 받는지" 보여주면 폼 작성률이 올라갑니다.

### 2. 참고 정보 탭 퀴즈 (3문항)
Appical이 프리보딩의 핵심 축으로 꼽는 "인터랙티브 콘텐츠".
읽고 → 확인하는 학습 루프를 만듭니다.

### 3. 체크리스트 서버 저장
현재 localStorage라 같은 링크라도 기기를 바꾸면 진행률이 날아갑니다.
(배송지 폼 제출 여부는 이미 GAS/Sheets에 저장되어 기기 간 유지됨 — 할 일
체크리스트만 아직 로컬에만 남는 상태)
